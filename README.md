# PRAD_Pathomics
This study developed and validated machine learning-based transcriptomic (MAPRS) and digital pathology (PSpc) risk scores for predicting recurrence and progression in prostate cancer after radical prostatectomy. These signatures showed superior prognostic performance and may guide personalized therapy.
